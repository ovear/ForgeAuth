ForgeAuth
=========

A simple Authentification system for Forge. Inspired by AuthMe

## How to build

For Minecraft 1.7.10 use branch `master`

For Minecraft 1.8 use branch `1.8`

Open a terminal

For Linux and Mac OS X
```
./gradlew build
```

For Windows
```
gradlew build
```

The jar will generated in build/lib/
